DNA MATCH
----------

Requisitos:
- Python 3.8 ou superior
- Biblioteca PPlay (instale com: pip install pplay)
- Biblioteca Pygame (instale com: pip install pygame)

Como jogar:
1. Extraia o conteúdo do arquivo zip
2. Abra o terminal na pasta extraída
3. Execute o seguinte comando:
   python main_dna.py

Objetivo:
Formar os pares corretos da molécula de DNA (A-T, C-G, A-U)
Você pontua por cada par correto formado. Dica: não deixe as bases erradas acumularem!
